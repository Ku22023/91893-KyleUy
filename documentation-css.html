CSS Examples and Explained

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages â€” for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

Selectors:

Selectors are patterns used to select the HTML elements you want to style.

#Element Selector

p {


color: blue;


}



#Class Selector

.my-class {


background-color: yellow;


}



#ID Selector

#my-id {


font-size: 20px;


}

Class vs ID:

Classes and IDs are attributes used to target HTML elements with CSS.

  • Class: Can be used on multiple elements. Use a dot (.) in CSS to select elements with a specific class.
  • ID: Should be unique to a single element per page. Use a hash (#) in CSS to select an element with a specific ID.

<div class="box">

I am a box with a class.

</div>


<p id="intro">

I am a paragraph with an ID.

</p>



/* CSS */

.box {


color: white;


}



#intro {


font-weight: bold;


}

Box Model:

The CSS box model is a container that surrounds every HTML element. It consists of: content, padding, border, and margin.

/* CSS Example */

.element {


width: 300px;


padding: 20px;


border: 5px solid red;


margin: 10px;


}

Margin:

Margin is the space outside the border of an element. It's used to create space between elements.

/* CSS Example */

.element {


margin-top: 10px;


margin-right: 20px;


margin-bottom: 15px;


margin-left: 5px;


}

Padding:

Padding is the space inside the border of an element, between the border and the content.

/* CSS Example */

.element {


padding-top: 10px;


padding-right: 20px;


padding-bottom: 15px;


padding-left: 5px;


}

Border:

The border property sets the border of an HTML element. You can specify the style, width, and color of the border.

/* CSS Example */

.element {


border-style: solid;


border-width: 2px;


border-color: black;


}

Content:

The content is the actual text, images, and other media inside the HTML element. The size of the content area can be affected by properties like width and height.

/* CSS Example */

.element {


width: 200px;


height: 100px;


background-color: lightblue;


text-align: center;


}



<div class="element">

My Content

</div>

Background:

The background property is used to set the background effects for an element, such as color or image.

/* CSS Example */

.element {


background-color: #ffffff;


background-image: url('image.png');


}

Flexbox:

CSS Flexbox is a layout module that makes it easier to design flexible and efficient layouts, especially for one-dimensional content arrangement (in rows or columns).

/* CSS for a flex container */

.container {


display: flex;


flex-direction: row; /* or column */


justify-content: center; /* Align items along the main axis */


align-items: center; /* Align items along the cross axis */


}
