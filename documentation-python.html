Python Examples and Explainations

What is Python?

Python is a popular and versatile programming language known for its readability and ease of use. It's used in web development, data science, artificial intelligence, and more.

Print (Show Output):

The print() function is used to display output to the console.

#Display the text "Hello world!"

print("Hello world!")



#Display the value of a variable

message = "hi"

print(message)

Input (Get User Data):

The input() function allows you to get input from the user.

#Get the user's name

name = input("Enter your name: ")

print(f"Hello, {name}")



#Get the user's age (input is always a string)

age_input = input("Enter your age: ")


age = int(age_input)


print(f"You are {age}, years old.")

Variables (Store Data):

Variables are used to store data values. You can assign different types of data to variables.

name = "Taichi"


age = 30


price = 99.99


is_student = False

Data Types:

Python has several built-in data types, including:

#String (text)


message = "Hello"



#Integer (whole numbers)


count = 10



#Float (decimal numbers)


pi = 3.14159



#Boolean (True or False)


is_valid = True



#List (ordered collection)


fruits = ['apple', 'banana', 'kiwifruit']

If/Else Statements (Make Decisions):

if, elif (else if), and else statements are used to execute different blocks of code based on conditions.

age = 20

if age >= 18:


print("You are an adult.")


else:


print("You are not yet an adult.")




score = 75


if score >= 90:


grade = "A"


elif score >= 80:


grade = "B"


else:


grade = "C"


print(f"Your grade is: {grade}")

Loops (Repeat Actions):

Loops allow you to execute a block of code multiple times.

For Loops:

for loops are used to iterate over a sequence (like a list, tuple, or string) or other iterable objects.

#Iterating over a list

fruits = ["apple", "banana", "cherry"]


for item in fruits:


print(item)



#Using range() to iterate a specific number of times

for i in range(3): #Generates numbers 0, 1, 2


print(f"Number: {i}")



#Iterating over a string

message = "Python"


for char in message:


print(char)

While Loops:

while loops continue to execute a block of code as long as a specified condition remains true. Be careful to avoid infinite loops!

count = 0


while count < 5:


print(f"Count is: {count}")


count += 1 #Important: Increase the counter to eventually exit the loop




#Using a boolean flag to control the loop


running = True


while running:


user_input = input("Type 'stop' to exit: ")


if user_input == "stop":


running = False


else:


print(f"You typed: {user_input}")

Lists (Ordered Collections):

Lists are ordered, changeable collections of items. They are defined using square brackets [].

my_list = [10, "hello", 3.14, True]




#Modifying items


my_list[0] = 20


print(my_list)

# Output: [20, "hello", 3.14, True]


#Adding items

my_list.append("world")


print(my_list)

#Output: [20, "hello", 3.14, True, "world"]

Functions (Reusable Code):

Functions are blocks of reusable code that perform a specific task. You can define your own functions.

#Define a function that greets a person

def greet(name):


print(f"Hello, {name}")



#Call the function

greet("Charlie")



#Define a function that adds two numbers and returns the result

def add(x, y):


return x + y


result = add(5, 3)


print(f"The sum is: {result}")
